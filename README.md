## 💡 Learning Points (학습 내용)
강의를 따라가며 단순히 코드를 복사하는 것에 그치지 않고, 다음과 같은 기술적 원리를 중점적으로 학습했습니다.

1. **고급 스크롤 인터랙션 구현**
   - Canvas와 자바스크립트를 활용한 고해상도 비디오/이미지 시퀀스 제어
   - 스크롤 진행도(Scroll Ratio)에 따른 요소의 투명도 및 위치 변화 계산
2. **시맨틱 마크업과 웹 표준**
   - 접근성을 고려한 태그 활용 및 애플 특유의 깔끔한 코드 구조 학습
3. **디테일한 CSS 스타일링**
   - 가독성 높은 타이포그래피 설정 및 정교한 여백(Margin/Padding) 시스템 적용

## 🛠 Tech Stack
- **Languages:** HTML5, CSS3, JavaScript (ES6+)
- **Tools:** GitHub Pages (Deployment)

## ✨ Key Features
- **Video Sequence Scroll:** 스크롤 위치에 맞춰 부드럽게 재생되는 영상 애니메이션
- **Sticky Layout:** 특정 구간에서 요소가 고정되며 설명이 나타나는 UI
- **Responsive Design:** 모바일부터 데스크탑까지 끊김 없는 사용자 경험 제공
